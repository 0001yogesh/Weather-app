# Weather Forecast App

A basic weather forecasting application UI built using HTML, CSS, and JavaScript.
(Current version works without external APIs.)

## Features
- City-based weather search (testing mode)
- Clean and responsive UI
- JavaScript-driven logic
- Ready for API integration

## Future Enhancements
- Real-time weather data using API
- Hourly & daily forecast
- Location-based weather
- Severe weather alerts

## Technologies Used
- HTML5
- CSS3
- JavaScript
